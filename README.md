# Gym Supplement Store

## Project Description

This project is a responsive frontend web application for a gym supplement store.
It was built using ReactJS and Bootstrap.

The website allows users to:

- Browse gym supplements
- Navigate between pages
- View responsive design on desktop and mobile

---

## Technologies Used

- ReactJS
- Bootstrap
- React Router DOM
- Vite
- Git & GitHub
- Netlify

---

## Pages

- Home
- About
- Products
- Contact

---

## Features

- Responsive Design
- Navigation Bar
- Hero Section
- Product Cards
- Mobile Friendly Layout

---

## Setup Instructions

1. Clone the repository
2. Open terminal
3. Run:

```bash
npm install
npm run dev
```
